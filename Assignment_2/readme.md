# Visual Entailment Analyzer

### Instructions for Evaluation:
Because machine learning model weights exceed GitHub's 100MB file limit, the `.pth` files are hosted safely on Google Drive.

**To run this project:**
1. Download the code from this repository.
2. Download the model weights from this link: (https://drive.google.com/drive/folders/1G8HXVayxYRuoW8L-69jry8BzTed1pGZP?usp=sharing)
3. Place the `.pth` files in the exact same folder as `app2.py`.
4. Run the Streamlit app.
